# vibnet
Repository for VibNet

A mobile application for accessible body stimulation.

Problem we solved: Limited solutions to solve sensorial imbalance, including but not limited to: anxiety, insomnia, disorintation, fatigue, anger, irritability, mood swings, restlessness, agitation, confusion, social withdrawal.
